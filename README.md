# Psychophysiological Methods in Cognitive Psychology

**Open Educational Resource**

**Author:** Dr. Yang, Yu-Fang  
**Institution:** Free University of Berlin

## About

Free textbook covering six psychophysiological measurement methods used in cognitive psychology research:

**EEG** · **EOG** · **EMG** · **EDA** · **ECG** · **EGG**

Designed for master's students and researchers learning these techniques.

## Chapters

1. [Electroencephalography (EEG)](chapters/eeg.md)
2. [Electrooculography (EOG)](chapters/eog.md) 
3. [Electromyography (EMG)](chapters/emg.md) 
4. [Electrodermal Activity (EDA)](chapters/eda.md) - Coming soon
5. [Electrocardiography (ECG)](chapters/ecg.md) - Coming soon
6. [Electrogastrography (EGG)](chapters/egg.md) - Coming soon

## For Students

Each chapter explains the physiological basis, measurement techniques, and research applications. Chapters include practical examples and reference current research.
