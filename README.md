# Psychophysiological Methods in Cognitive Psychology

**Open Educational Resource**

Dr. Yang, Yu-Fang | Free University of Berlin

---

## About

Free textbook covering six psychophysiological measurement methods used in cognitive psychology research:

**EEG** · **EOG** · **EMG** · **EDA** · **ECG** · **EGG**

Designed for master's students and researchers learning these techniques.
